# URL to Markdown: Capture Modes & Adapters

Detailed documentation on the various ways to capture web content and the specialized site adapters available.

## Capture Modes
- **Headless (Default)**: Fast, non-visible Chrome.
- **Wait-for Interaction**: Visible Chrome, waits for login or CAPTCHA resolution.
- **Wait-for Force**: User manual control (press Enter to capture).

## Built-in Adapters
| Adapter | Target |
|---------|--------|
| `x` | Tweets, Threads, X Articles |
| `youtube` | Transcripts, Chapters, Metadata |
| `hn` | Hacker News comments with nesting |
| `generic` | Generic pages (Defuddle/Readability) |
