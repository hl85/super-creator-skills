# ChatGPT 网页版生图操作指南

## 页面识别特征

ChatGPT 页面（https://chatgpt.com/）的关键元素：
- 输入框：`role: textbox, name: "与 ChatGPT 聊天"`
- 发送按钮：`role: button, name: "发送提示"`（生成中变为"停止回答"）
- 图片反馈按钮：`role: button, name: "喜欢此图片"` / `"不喜欢此图片"`
- 更多操作：`role: button, name: "更多操作"`（在图片附近）
- 图片元素：`role: img, name: "已生成图片：[描述]"`

## 前置检查

1. 使用 `browser_tabs` 找到 ChatGPT 标签页
2. 调用 `browser_snapshot(interactive: true)` 检查状态：
   - 如果看到"登录"按钮 → 提示用户先在浏览器中登录
   - 如果看到输入框和"准备好了，随时开始" → 就绪

## 详细步骤

### 步骤 1：选择模型（可选）

如果需要特定模型（如 GPT-4o）：
```
1. 找到"模型选择器"按钮
2. browser_click 打开选择器
3. snapshot 找到目标模型选项
4. browser_click 选择
```

### 步骤 2：聚焦输入框并输入提示词

```
1. 找到输入框 ref（"与 ChatGPT 聊天" textbox）
2. browser_click(ref: 输入框ref)
3. 输入提示词：
browser_type(
  ref: 输入框ref,
  text: "生成一张[描述]图片，DALL-E风格，[其他要求如背景、色彩等]"
)
```

**提示词建议：**
- ChatGPT 中英文提示词都可以
- 可以明确要求"使用 DALL-E 生成"
- 描述尽量具体：风格、场景、色彩、构图

### 步骤 3：发送并等待生成

```
1. 找到"发送提示"按钮
2. browser_click(ref: 发送按钮ref)
3. 等待循环：
   - 等待 5 秒
   - browser_snapshot()
   - 检查"停止回答"按钮是否消失
   - 检查是否出现"喜欢此图片"/"不喜欢此图片"按钮
   - 检查是否出现 img 元素
   - 超时时间：90秒（DALL-E 生图可能稍慢）
```

### 步骤 4：下载图片

ChatGPT 没有 Gemini 那样直接的下载按钮，使用以下方式：

**方式 1：点击图片查看大图后下载**
```
1. browser_click(ref: 图片img元素ref) 打开大图预览
2. snapshot 查找下载按钮或右键菜单
3. 点击下载选项
```

**方式 2：通过"更多操作"菜单**
```
1. 找到图片附近的"更多操作"按钮（e19）
2. 注意：该按钮在图片完全加载前可能 pointer-events: none，需要等待
3. browser_click(ref: 更多操作按钮ref)
4. 在弹出菜单中查找"下载"选项
5. 点击下载
```

**方式 3：JavaScript 获取图片 URL（备选）**
```
使用 browser_evaluate 执行脚本获取图片 src，然后：
1. 如果是 blob URL，提示用户手动右键保存
2. 如果是直接 URL，可以用其他方式下载
```

**方式 4：提示用户手动保存（保底方案）**
如果自动下载遇到困难，直接提示用户：
"图片已生成，请右键点击图片选择'图片另存为'保存到本地。"

## DALL-E 提示词模板

### 写实照片风格
```
生成一张[主题]的照片，[场景描述]，
专业摄影风格，8k分辨率，自然光线，细节丰富，
photorealistic, high detail, professional photography.
```

### 插画风格
```
生成一张[主题]的插画，[风格描述如：卡通/水彩/扁平设计]，
[色彩方案]，[构图描述]，适合用于[用途如：文章配图/海报]，
illustration style, clean design, high quality.
```

### 公众号封面图
```
为文章《[文章标题]》生成一张封面图，
[文章核心内容摘要]，
风格简洁大气，适合微信公众号封面尺寸（2.35:1），
色彩协调，有视觉吸引力。
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| "更多操作"按钮点不动 | 等待图片完全加载，按钮状态从 disabled 变为可用 |
| 生成速度慢 | DALL-E 3 通常需要 20-60 秒，耐心等待 |
| 图片不显示 | 刷新页面重试，或检查网络连接 |
| 提示"生成图片失败" | 调整提示词，避免敏感内容；重试一次 |
| 无法自动下载 | 直接提示用户右键手动保存，这是最可靠的方式 |
| 只生成了1张图 | 在提示词中明确说"生成2张/4张不同风格的图片" |
