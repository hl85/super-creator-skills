## 风格组合指南

### 推荐组合方式

1. **单风格**（推荐）：选择 1 个主风格直接使用
2. **技法 + 色调**：如 `watercolor` + `warm` 色调、`chalkboard` + `elegant` 色调
3. **避免冲突**：不要同时叠加多个技法类风格（如 watercolor + pixel-art）

### 按场景推荐

| 场景 | 推荐风格 |
|------|---------|
| 技术博客/教程 | notion, blueprint, technical-schematic |
| 公众号封面 | elegant, editorial-infographic, bold-editorial, dark-atmospheric |
| 小红书图文 | cute, fresh, morandi-journal, playful, study-notes |
| 知识科普 | scientific, sketch-notes, vector-illustration |
| 情感/故事 | fantasy-animation, storybook-watercolor, warm |
| 数据/信息图 | blueprint, editorial-infographic, subway-map |
| 儿童/趣味 | kawaii, lego-brick, claymation, flat-doodle |
| 复古/怀旧 | vintage, retro, aged-academia, retro-pop-grid |

## 文件位置

所有风格定义文件位于：`references/styles/{style-id}.md`

每个文件包含标准化的结构：
- **Atmosphere**：氛围描述
- **Color Palette**：调色板（含 Hex 值）
- **Visual Elements**：核心视觉元素
- **Style Rules**：Do/Don't 规则
- **Example Prompt**：可直接使用的英文提示词模板
- **Best For**：最佳适用场景
