# claymation

3D clay figure aesthetic with stop-motion charm

## Atmosphere

Squishy, malleable clay forms feel warm to the touch. Fingerprint indentations and slight imperfections reveal the handmade process. Soft studio lighting casts gentle shadows, and every rounded shape exudes playful, tactile charm.

## Color Palette

- Primary: Saturated clay colors - bright but slightly muted
- Background: Neutral studio backdrop, soft gradients
- Accents: Complementary clay colors, shiny highlights

## Visual Elements

- Clay/plasticine texture on all objects
- Fingerprint marks and imperfections
- Rounded, sculpted forms
- Soft shadows
- Stop-motion staging
- Miniature set aesthetic

## Typography

- Extruded clay letters
- Dimensional, rounded text
- Playful and chunky
- Embedded in clay scenes

## Style Rules

### Do

- Apply visible clay/plasticine texture to all objects
- Include subtle fingerprint marks and surface imperfections
- Use rounded, sculpted forms with soft edges
- Add soft, diffused studio lighting and gentle shadows
- Stage elements like a stop-motion set with miniature props

### Avoid

- Creating sharp, angular edges or perfect geometry
- Using smooth, textureless surfaces
- Adding harsh, high-contrast lighting
- Including photographic elements or realistic textures
- Using thin, delicate typography

## Example Prompt

A playful process infographic explaining how to make cookies, rendered in claymation style: 3D clay figures and objects with visible fingerprint marks, squishy rounded forms, soft studio lighting and gentle shadows, miniature kitchen set aesthetic, extruded clay letters, saturated but slightly muted clay colors, 16:9 aspect ratio

## Best For

Playful explanations, children's content, stop-motion narratives, friendly processes
