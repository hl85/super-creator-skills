# pixel-art

Retro 8-bit gaming aesthetic

## Atmosphere

The faint glow of a CRT screen hums with nostalgic energy. Blocky pixels form iconic shapes against a dark background, while limited color choices evoke the charm of classic gaming consoles. The texture feels crisp and digital, with visible scanlines and pixel-perfect edges.

## Color Palette

- Primary: Limited palette - NES/SNES colors
- Background: Black or dark blue, scanlines optional
- Accents: Bright pixel highlights, CRT glow

## Visual Elements

- Visible pixel grid
- Limited color count per sprite
- 8-bit or 16-bit style
- Retro game UI elements
- Pixel-perfect edges
- Dithering for gradients

## Typography

- Pixel fonts
- Blocky letterforms
- Game UI style text
- Score/stat display style

## Style Rules

### Do

- Use visible pixel grid and pixel-perfect edges
- Stick to limited retro color palettes (NES/SNES style)
- Use dithering for gradients and shading
- Include retro game UI elements where appropriate
- Maintain 8-bit or 16-bit aesthetic

### Avoid

- Using smooth gradients or anti-aliased edges
- Adding photorealistic elements or photographs
- Using unlimited or modern color palettes
- Creating rounded or organic shapes instead of blocky forms
- Omitting visible pixel structure
