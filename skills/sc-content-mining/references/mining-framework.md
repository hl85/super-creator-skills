# Mining Framework: 认知错位提取方法

## 提取结构

对每份内容源，提取以下结构：

```yaml
选题:
  标题: "..."              # 拟定的社交媒体标题
  来源: 内容标识           # 来源内容/学员/章节
  认知错位:
    读者以为: "..."         # 读者/受众原本以为的
    真相: "..."             # 内容中指出的真相
  传播力: ⭐⭐⭐            # 1-5 星
  受众面: ⭐⭐⭐            # 1-5 星
  情绪共鸣: ⭐⭐⭐          # 1-5 星
  金句: "..."              # 可直接用于封面的原话
  来源段落: "..."           # 内容中的原文出处
```

## 评分标准

```
传播力 = 认知颠覆强度 × 受众覆盖面 × 情绪共鸣度
```

### 认知颠覆强度
| 星级 | 标准 | 示例 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 完全反常识 | "你以为面试考技术？其实他在赌你潜力" |
| ⭐⭐⭐⭐ | 部分反常识 | "刷题有用，但不是你以为的那种有用" |
| ⭐⭐⭐ | 新视角 | "你没想过的角度" |

### 受众覆盖面
| 星级 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 目标领域全人群 |
| ⭐⭐⭐⭐ | 特定细分群体 |
| ⭐⭐⭐ | 特定阶段/场景 |

### 情绪共鸣度
| 星级 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 焦虑+恐惧（"你可能正在犯这个错"） |
| ⭐⭐⭐⭐ | 共鸣+释然（"原来不只我这样"） |
| ⭐⭐⭐ | 好奇+求知（"这个角度没想过"） |

## 常见认知错位模式库

以下为面试辅导行业的示例模式，各行业可根据自身领域扩展。

### 📌 示例：面试辅导行业

| 错位类型 | 学员以为 | 真相 | 纪要来源 |
|---------|---------|------|---------|
| 面试本质 | 面试=技术考试 | 面试官在赌你潜力 | 沈安迪、H同学 |
| 精力分配 | 90%精力准备技术面 | 技术面只是入场券 | 沈安迪、H同学 |
| 经验价值 | 经验越多越有优势 | 大厂螺丝钉困境 | 沈安迪 |
| 刷题万能 | 刷题能进大厂 | 刷题是体力模式，年龄越大越不灵 | 沈安迪 |
| 项目包装 | 罗列"我用了什么" | 要讲"我解决了什么问题" | H同学、沈安迪 |
| 全栈vs深度 | 转全栈才是出路 | 前端深度（工程化）更有价值 | H同学 |
| 晋升vs跳槽 | 跳槽涨薪快 | 缺晋升经历=缺深度复盘能力 | 沈安迪 |
| 能写不能讲 | 我技术不够好 | 缺结构化表达能力 | H同学 |

### 各行业认知错位模式扩展思路

| 行业 | 错位类型 | 受众以为 | 真相 |
|------|---------|---------|------|
| **职场发展** | 努力与回报 | 只要努力就能升职加薪 | 升职靠的是不可替代性+向上管理 |
| **职场发展** | 跳槽频率 | 频繁跳槽简历就废了 | 有清晰成长路径的跳槽反而是加分项 |
| **家庭教育** | 孩子成绩 | 报班越多成绩越好 | 学习习惯和内驱力才是核心 |
| **家庭教育** | 听话与独立 | 听话的孩子更有出息 | 有独立思考能力的孩子走得更远 |
| **健康养生** | 减肥方法 | 少吃多动就能瘦 | 激素平衡和睡眠质量影响更大 |
| **健康养生** | 运动强度 | 运动越久效果越好 | 适度运动+长期坚持 > 短期高强度 |
| **个人理财** | 理财门槛 | 没钱就不用理财 | 理财是一种思维方式，从第一笔收入就该开始 |
| **个人理财** | 风险认知 | 存银行最安全 | 通胀会悄悄吃掉你的购买力 |
