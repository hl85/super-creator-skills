# Format Recommendations: 呈现形式建议

挖掘完成后，为每个选题推荐呈现形式。**每条建议必须含简洁效果描述**。

## 呈现形式选项

### sc-xhs-images 风格（11 种）

| 风格 | 效果描述 | 适合选题类型 |
|------|---------|------------|
| **bold** | 红黑高对比，大字 3D 阴影，感叹号+箭头，视觉冲击最强 | 反认知、避坑警告 |
| **chalkboard** | 彩色粉笔黑板风，教育感强，适合知识点拆解 | 教程、清单、知识点 |
| **cute** | 甜美可爱，粉色系，适合轻松话题 | 种草、生活方式 |
| **fresh** | 清新自然，浅色调，舒适阅读 | 日常分享、工具推荐 |
| **minimal** | 极简留白，黑白灰，高端感 | 高端职场、深度思考 |
| **notion** | 极简线条画，手绘抖动效果，知识感 | 干货知识卡片、清单 |
| **pop** | 波普艺术，撞色，视觉活泼 | 趣味科普、对比 |
| **retro** | 复古风，怀旧色调，有故事感 | 个人经历、故事类 |
| **screen-print** | 丝网印刷，色块平涂，概念化 | 抽象概念、金句卡 |
| **study-notes** | 手写笔记风，照片质感，真实感 | 学习笔记、复盘 |
| **warm** | 暖色调，温馨，有温度 | 个人感悟、情感共鸣 |

### 布局（8 种）

| 布局 | 效果描述 | 适合内容结构 |
|------|---------|------------|
| **sparse** | 大量留白，单一焦点，最大冲击 | 封面、金句卡 |
| **balanced** | 上下平衡，标题+内容，标准布局 | 通用内容页 |
| **dense** | 信息密集，网格结构，干货满 | 知识卡片、清单 |
| **list** | 垂直列举，数字编号，层级清晰 | 排行榜、步骤、清单 |
| **comparison** | 左右对比，视觉反差，强调差异 | 你以为vs真相、前后对比 |
| **flow** | 连接节点，方向流动，流程感 | 流程图、递进关系 |
| **mindmap** | 中心放射，分支展开，发散感 | 概念地图、知识体系 |
| **quadrant** | 四象限网格，分类对比 | SWOT、优先级矩阵 |

### 其他视觉 skill

| Skill | 效果描述 | 适合场景 |
|-------|---------|---------|
| **sc-cover-image** | 文章封面，5 维度组合，10 色板 | 公众号/博客封面 |
| **sc-article-illustrator** | 文章配图，自动识别插图位置 | 长文章配图 |

## 推荐格式

挖掘完成后，为每个选题输出：

```markdown
### 选题 N: [标题] 传播力 ⭐⭐⭐⭐⭐

**认知错位**: 学员以为 X → 真相是 Y
**金句**: "..."
**来源**: 沈安迪纪要

**建议呈现形式**:
- 风格: bold — 红黑高对比，大字3D阴影，视觉冲击最强，适合反认知标题
- 布局: comparison — 左右对比"你以为 vs 真相"，强化认知颠覆感
- 图片数: 6 张（封面+对比+3个核心点+行动清单）
```
